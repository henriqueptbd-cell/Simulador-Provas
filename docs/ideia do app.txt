# Estrutura Base — Banco de Questões de Português (Concursos)

## Objetivo

Estruturar um banco de questões modular em JSON para:

* app de simulados
* treino mobile
* aleatorização de questões
* filtros por tema
* revisão inteligente
* expansão futura
* integração com IA

---

# Estrutura de Pastas

```txt
/questions
  /portugues
    interpretacao.json
    pontuacao.json
    concordancia.json
    crase.json
    regencia.json
    classes_gramaticais.json
    ortografia.json
```

---

# Padrão de IDs

## Formato

```txt
PT-TEMA-NUMERO
```

## Exemplos

```txt
PT-IN-001  -> interpretação
PT-PO-001  -> pontuação
PT-CO-001  -> concordância
PT-CR-001  -> crase
PT-RE-001  -> regência
PT-CL-001  -> classes gramaticais
PT-OR-001  -> ortografia
```
